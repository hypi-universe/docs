Tables

  1. Click Database Icon on the top right corner of the Stories board to add Tables.

  1. You will be able to create a Table adding the entries visually through UI. Click on the the +Create A New Table or Database icon to add a new table.

  1. Name the table based upon your data model. Click +Add New Field to add fields/columns to the table.

  1. Using this UI, you will be able to enter various data types like string, integer, float, ID, Boolean, etc. Enter the field name and select the type of the field.

  1. You can declare an array of a field type by clicking Allow Multiple Values. You can also make a field mandatory by clicking Required.

  1. You may select the field having any table as a data type. Select the data table from the drop-down list to link the table.

  1. Click the Save Field button to save the field.

  1. Table entries get saved in the table.

  1. If you need to edit an entry, hover over the field and click the settings icon.

  1. You may delete the entries by clicking the delete icon. This option is available only under draft versions. For published versions, the fields will remain read-only.

  1. Hypi auto-generates common fields for all types for easy filtering and sorting. Read more about them here.
