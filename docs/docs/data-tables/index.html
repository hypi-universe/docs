Collections

Overviewâ€‹

After adding User stories, you will have an idea of what kind of data needs to be stored. Based on that, you can design your data models. From the App Designer board, you can go to Collections to add your data models/tables.

Data Tablesâ€‹

  1. Click Database Icon on the left-hand side of the App Designer Board to add Tables.

  1. You will be able to create a Table adding the entries visually through UI. Click on the the New Table or Database icon to add a new table.

  1. Name the table based upon your data model. Click +Add Field to add fields/columns to the table.

  1. Using this UI, you will be able to enter various data types like string, integer, float, ID, Boolean, etc. Enter the field name and select the type of the field.

  1. You can declare a list of a field type by clicking This is a list of []. You can also make a field mandatory by clicking This is a required field.

  1. You may select the field having any table as a data type. Select the data table from the drop-down list to link the table.

  1. Click the Save button to save the field.

  1. Table entries get saved in the table. Currently you can't edit or delete the entries.

  1. Hypi auto-generates common fields for all types for easy filtering and sorting. Read more about them here.
